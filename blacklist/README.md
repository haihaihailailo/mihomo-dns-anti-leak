# Android 分应用代理黑名单

这些文件用于 Clash Mi / Android 分应用代理的黑名单模式。

## 原则

- 黑名单中的应用完全绕过 Android VPN，直接使用系统网络。
- 只放确实需要绕过 VPN 的超级 App / 小程序入口，不要把所有国内 App 都加入黑名单。
- 修改黑名单后，需要停止并重新启动 VPN，再完全退出相关 App 重新测试。
- 不要把 Clash Mi 自身 `com.nebula.clashmi` 加入黑名单。
- 不要开启 Android 系统的“无 VPN 时阻止连接 / Block connections without VPN”，否则黑名单 App 可能无法联网。

## 当前默认黑名单

| 应用 | 包名 | 原因 |
|---|---|---|
| 微信 | `com.tencent.mm` | 小程序、H5、第三方商家页面属于混合入口，直接绕过 VPN 更稳定 |
| 支付宝 | `com.eg.android.AlipayGphone` | 小程序、H5、支付与第三方服务属于混合入口，直接绕过 VPN 更稳定 |

## 暂不默认加入

QQ、云闪付、银行、数字人民币、政务、钱包类 App 暂不默认加入。只有实际出现加载失败、定位异常、支付失败或风控问题时，再按设备单独加入。

## 配置配合

微信和支付宝由分应用黑名单绕过 VPN，因此主配置中的对应 `PROCESS-NAME` 规则应注释。Chrome、Android System WebView、雨见等混合入口交给域名规则分流。OPPO 浏览器继续固定到 `国内服务`。
